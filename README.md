# LLM 기반 업무 자동화 프롬프트 엔지니어링 프로젝트

## 프로젝트 개요

본 프로젝트는 **서비스 시장 분석(TAM, SAM, SOM 산출)** 업무에 가장 적합한 LLM을 선정하기 위한 비교 분석 프로젝트이다.

### 업무 과업

* 목적: 신규 서비스 시장 분석 자동화
* 타겟 사용자: 스타트업 기획자 및 예비 창업자
* 입력:

  * 서비스 설명
  * 타겟 고객
  * 수익모델
  * 시장 범위
* 출력:

  * TAM
  * SAM
  * SOM
  * 경쟁사 분석
  * 시장 진입 전략

## 제출 문서

### 1. 모델 비교 및 선정

* 파일명: `LLM 모델 비교 선정.md`
* 내용:

  * GPT-4o
  * Claude Sonnet
  * Gemini
  * 성능 비교
  * 환각 검증
  * 최종 모델 선정

### 2. 시스템 설계

* 파일명: `시스템 설계.md`
* 내용:

  * 시스템 프롬프트
  * 유저 프롬프트
  * 입력 템플릿
  * Few-shot 예시
  * 프롬프트 개선(v1 → v2)

### 3. 실행 로그

* 파일명: `실행 로그.md`
* 내용:

  * 실제 대화 기록
  * 10턴 이상 대화
  * 중간 조건 변경 사례
  * 결과 검증
